Forgematica is the Forge port of **Litematica** — a building-blueprint mod. Load a `.litematic` schematic file and it projects a translucent **ghost hologram** of the build into the world, so you can copy it block by block. It comes with a material list, a verifier that checks your work, and area-selection tools for making your own schematics.

!!! info "Client-side only"
    Forgematica runs entirely on your own client — it just draws a hologram and guides you, and the server never sees it. **You only need it in your client**, and the pack handles the rest automatically.

<p class="sb-eyebrow">How you'll use it</p>

1. Press **`M`** to open the main menu, then load a `.litematic` file.
2. Place it in the world to create a **placement** — the ghost hologram appears where you put it.
3. Build by copying the hologram block by block.
4. Open the **Material List** (`M`, then `L`) to see what materials you still need.
5. Check your work with the **Verifier** (`M`, then `V`) — it marks each block as correct, missing or wrong.

!!! tip "Survival vs creative"
    In survival you'll mostly use the **hologram**, **material list** and **verifier** to build by hand (plus Easy Place). The mass paste/fill/move tools are creative-only.
