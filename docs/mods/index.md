    <h1>Dimensions</h1>
  </div>

Realm Gates is built around **dimensions** — separate realms you travel between as you progress.
You start in **Aincrad** (Dimension 1), and more realms open up over time (the
**[Realm Gates](../custom/realmgates.md)** mod controls when each one unlocks). We're still pinning
each world/creature mod to the realm it belongs to, so some cards are tagged **Unassigned** for now.

Want the full list? **[All mods](all-dimensions.md)** has a friendly page for every mod in the pack —
each tagged by where it belongs (a specific **Dimension**, **All dimensions** for the universal tools
that follow you everywhere, or **Unassigned** if it hasn't been pinned to a realm yet).
