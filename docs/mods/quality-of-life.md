# Quality of life — *rank 0*

These mods are active from minute one. They don't gate progression — they make the whole game
easier to read, smoother to run, and more social.

## Information & UI

### Just Enough Items (JEI)
A searchable list of **every item and recipe** in the pack. Look up how to craft something, or
what an item is used for. Press the recipe key while hovering an item. *Your #1 tool for a modded
game.*

### Just Enough Resources (JER)
Extends JEI with **where things come from** — which ores spawn at which depths, mob drops, and
world-gen info.

### Just Enough Breeding
Extends JEI with **how to breed each animal** (what food, what conditions).

### Jade
Point at any block or creature and a small overlay shows **what it is**, its health, and the right
tool to mine it. The fastest way to learn a modded world.

### JourneyMap
A live **minimap** and full-screen map. Set waypoints, see where you've explored, and find your way
home. Essential with this many new biomes.

### AppleSkin
Shows **food saturation and hunger** details on the HUD, so you can tell how filling food really is.

### Controlling
Makes the **keybind menu searchable** — find and fix conflicting controls instantly. With this
many mods, you'll want it.

### Highlighter
Subtly **outlines the block you're looking at** so it's easier to target precisely.

## Performance (work silently in the background)

| Mod | What it does |
| --- | --- |
| **Entity Culling** | Skips rendering creatures/blocks you can't actually see → more FPS. |
| **FerriteCore** | Cuts the game's **memory** use. |
| **ImmediatelyFast** | Speeds up rendering of text, items, and UI. |
| **Clumps** | Merges nearby XP orbs into one, reducing lag at mob farms. |

## Looks & social

### FancyMenu
Powers the **custom main-menu theme** for summerBuddies.

### Enhanced Visuals
Adds **screen effects** (e.g. a fire/blood overlay when hurt) for a more immersive feel.

### Spawn Animations
Creatures **rise/animate into the world** when they spawn, instead of popping in.

### Simple Voice Chat
**Proximity voice chat** — players near you are louder, and the sound fades with distance. It also
provides the system that **[Voice Translate](../custom/voice-translate.md)** is built on. Set your
mic with the settings key (default **`V`**). See
[Voice & chat translation](../getting-started/voice-and-chat.md).

### No Chat Reports
A **privacy** mod that disables Minecraft's chat-reporting system. Nothing for you to configure.
