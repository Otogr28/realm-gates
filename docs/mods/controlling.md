With this many mods, the keybinding menu gets *long*. Controlling makes it **searchable** and helps you spot **conflicts**, so setting up your controls takes minutes instead of scrolling forever.

<p class="sb-eyebrow">What it adds</p>

- A **search bar** in the Controls menu — type an action or a key to find it instantly.
- A quick way to see and fix **conflicting keybinds**.
- A "free keys" filter to find buttons you haven't used yet.
